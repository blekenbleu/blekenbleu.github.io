COVID-19 in selected counties


This uses selections from Johns Hopkins Time Series CSV for COVID-19
Selections are based on myFIPS.csv, which was in turn extracted from FIPS county population data, e.g.
https://www2.census.gov/programs-surveys/popest/datasets/2010-2019/counties/totals/co-est2019-alldata.csv
FIPSpopJul2018.csv is a stripped CSV file with only state,county,population of US FIPS localities.

Processing uses bash scripts, gnuplot, and ImageMagick
The version of bash used is from Ruby on Windows 10.
I expect these scripts will require changes to work elsewhere,
e.g. Windows Subsystem for Linux, macOS or linux.

Extraction is mostly accomplished by frame.sh, which may be invoked standalone,
but is typically run by invoking covid.sh, which will download most recent Johns Hopkins Time Series .csv,
then invokes frame.sh, which eventually invokes gnuplot to generate a covid.png based on plot_covid.p,
which loads title.p and uses data.txt, both having first been generated by frame.sh.

animate.sh also invokes frame.sh to generate one suitably renamed covid.png for each day in the animation,
which are in turn rotated and combined into an animated gif, as displayed above, by magick convert.

Utility script cfg.sh is sourced by both animate.sh and covid.sh
to set environmental variables for paths to scripts, executibles and working subdirectory.
