OxyScope Help

Controls:  left to right. top to bottom
top row
Help - link to this page

Current property sample set statistics, updated every 3 seconds;
may not get plotted, depending on buttons to the right...

 REPLOT  - enabled by the next button

Manual / Auto Replot
  • when button displays Manual Replot,
    displayed plot remains until red REPLOT is selected
  • when button displays Auto Replot, plots update as new samples are collected
3 second refresh / Hold max X range
  • refresh: sample set can be plotted
  • Hold: only sample sets with greater range of X values will plot
Linear Fit
  • Disable:  only scatter plot
  • Enable:  adds leasts square fit line, also forced origin line for bipolar Y values
row 2: Properties XY plot
X, Y sample set pairs are plotted, when X value range satisfies criteria.

row 3:  Current plot statistics
typically reports slopes for fitted lines and Y intercept for line not thru origin

row 4:  Y and X properties being sample.
Copy names of interest from SimHub's Available properties window

bottom row:  filters (1 == no filtering)
Some properties depend on others, but with delayed effect.
Increasing filter for property provoking response may improve plotted correlation
