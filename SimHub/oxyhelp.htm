OxyScope Help

Controls:  top to bottom
top row:  left-to-right
Help - link to this page

Current property sample set statistics, updated every 3 seconds;
  may not get plotted, depending on buttons to the right...

 REPLOT  - enabled by the next button set to Manual

Manual / Auto Replot
  • when button displays Manual Replot,
    displayed plot remains until REPLOT is selected
  • when button displays Auto Replot,
    plots update as new samples are collected

refresh:  3 second / Hold max X range / Hold max R-squared
  • 3 second: all sample sets can be plotted
  • Hold max X: only sample sets with greater range of X values will plot
  • R-squared: only sets with better correlation to linear solution

Linear Fit
  • Disable:  only scatter plot
  • Enable:  adds leasts-squares fit line and monotonic cubic curve,
    also forced origin line for bipolar X,Y samples

row 2: Properties XY plot
X, Y sample set pairs are plotted, based on refresh setting

row 3:  XY plot property names
Copy names of interest from SimHub's Available properties window

rows 4, 5:  Current plotted property statistics
typically reports coefficients for fitted lines, cubic fits

bottom row:  filters (1 == no filtering)
Some properties affect others, but with delay.
Increasing filter value for property affecting response
  may improve correlation..
