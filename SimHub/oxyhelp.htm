OxyScope Help

Controls:  left to right. top to bottom
top row
Threshold slider:  1-99 (when X Threshold is enabled)
  • ignore X property values less than this slider setting
Manual Replot
  • when button displays Manual Replot,
    displayed plot remains until red REPLOT is selected
  • when button displays Auto Replot, plots update as new samples are collected
X Thresold displays value, controled by slider, for collecting samples

Disable / Enable Threshold
  • Disable accepts all non-negative property values
Linear Fit disabled / enabled
  • enabled plots least squares regressions with and without forced origin
second row
X Below slider
  • X and Y plot axes both auto-scale
  • slider values sets dynamic range, with minimum sample value < slider % of full scale
REPLOT button, when Manual Replot is selected in top row

Y Below slider:  as with X Below, 100% allows all non-negative samples

third row: Properties XY plot
X, Y sample pairs are plotted.
Both paired values must satisfy critereon to be collected.

row 4:  current status
typically reports slopes for fitted lines and Y intercept for line not thru origin

row 5:  Y and X properties being sample.
Copy names of interest from SimHub's Available properties window

bottom row:  filters (1 -- no filtering)
Some properties depend on others, but with delayed effect.
Increasing filter for property provoking response may improve plotted correlation
