SimHub

SimHub extracts telemetry as properties from driving sims,
coordinating events, actions and properties among plugins.
SimHub is popular for dashboards;
I use it for controls, fans, harness tensioning and tactile feedback.

controls
  • Assetto Corsa uses DirectInput controllers,
    which can be emulated using vJoy.
    • MIDIio SimHub plugin converts among
      • MIDI control surface change signals
      • vJoy emulated DirectInput signals
      • SimHub properties to/from other plugins
        or derived from game telemetry.
  • Fake8 SimHub plugin converts between SimHub properties
    and (e.g. STM32 or ESP32) Arduino module COM messages
    via SimHub Custom Serial devices.
  • InterceptMouse is an experiment for converting
    second mouse events to SimHub properties.
SimHub ShakeIt profiles use properties to control:
  • fans and harness tensioning via Arduino
  • tactile feedback
    • ShakeIt modulates tactile amplitudes and/or frequencies,
      but not independently.
    • MIDIio can independently modify tactile signals
      via MIDI VST.
    • ShakeIt profiles can also export custom properties.
