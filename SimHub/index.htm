SimHub

SimHub extracts telemetry from many driving games.
Many folks use SimHub for dashboards;
I use it for controls, fans, harness tensioning and tactile feedback.

controls
  • Assetto Corsa works with DirectInput devices,
    which can be emulated using vJoy.
  • MIDIio SimHub plugin enables converting, among other things, MIDI control surface inputs to vJoy DirectInput signals
  • Fake8 SimHub plugin enables, among other things,
    converting Arduino COM messages to vJoy DirectInput signals
  • InterceptMouse is an experiment to capture inputs
    from a single mouse for redirection e.g. to vJoy.
fans and harness tensioning
  • SimHub ShakeIt uses game telemetry to control fan speeds
    and harness tensioning via Arduino.
    SimHub ShakeIt profiles can also export custom properties generated by JavaScript from telemetry.
  • Fake8 SimHub plugin enables, among other things,
    controlling these by a STM32 or ESP32 module
    via SimHub Custom Serial devices.
  • SimHub ShakeIt and Custom Serial profiles
          Custom Serial profile atlas
          ShakeIt profile descriptions
tactile feedback
  • SimHub ShakeIt uses game telemetry to modulate tactile
    amplitude and/or frequency, but not independently.
  • MIDIio SimHub plugin enables, among other things, independently modifying tactile signal amplitudes
    from ShakeIt propertiesi via MIDI VST.
