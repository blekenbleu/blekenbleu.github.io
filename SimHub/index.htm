SimHub

SimHub extracts telemetry as properties from driving sims,
coordinating events, actions and properties among plugins.
SimHub is popular for dashboards, haptics and motion;
I use it for controls, fans, harness tensioning and tactile feedback.

Sim Controls
  • Assetto Corsa uses DirectInput controllers,
    which can be emulated using vJoy.

  • Direct-Drive-harness-tension-tester
  • Fake8 SimHub plugin converts between SimHub properties

  • @sierses' SimHaptics with some bug fixes
  • XtraMouse dedicates a second (or third) mouse to SimHub

SimHub ShakeIt profiles use properties to control
  • tactile feedback
    • modulating tactile amplitudes and/or frequencies.
  • ShakeIt profiles can also export custom properties
    that can control e.g. fans and harness tensioning
    via Custom Serial profiles and STM32 Arduino
  • MIDIio can separately route property values
    e.g. via vJoy and/or MIDI VST.
