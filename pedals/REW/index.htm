    Intro

    REW is not for only rooms [click for details] REW is mostly used to help locate and tune subwoofers,
    mostly by measuring and displaying audio responses.
    Frequency response peaks can be mitigated by equalizing,
    but sharp dips typically result from cancellations AKA destructive interference,
    which are better addressed by relocating a single subwoofer
    or changing phase relations among multiple subwoofers, also possibly by relocation.
    The same techniques will apply for tactile transducers,
    keeping in mind that sound travels faster thru solids than air,
    so that substantial changes in phase relationships are more easily done electronically.
