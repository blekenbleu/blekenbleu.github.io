Windows audio basics

Windows 11 sound routing

Windows sound output does not play default input;  some application must route it.
Except for defaults, Windows 11 configures sound devices for exclusive access.
Windows settings could, but fails to warn when multiple programs are misconfigured
to use the same device having exclusive access set.
Consequently, any not-default input (e.g. text-to-voice) output must be somehow split
to feed both e.g. Somnium Space VR and default output.
System > Sound > Sound Mixer,
where Sound Mixer requires scrolling nearly to the bottom of System > Sound.
The v to the right of each App exposes Output and Input device selection.

VoiceMeeter patches

Reconfigured VoiceMeeter AUX as default Windows sound default output device
and VoiceMeeter VAIO as Windows sound default input.
source input X-Fi CABLE In Denon Main OUT
default in		 Aux OUT

INPUT 2

Translated: Text to Voice outputs to VAIO (default input),
used e.g. by Somnium but also passed to X-Fi for headphones and Denon for surround.
AUX is default output, also passed to X-Fi and Denon (but not Somnium).
