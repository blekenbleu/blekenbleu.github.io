Windows 11 sound routing

Windows sound output does not play default input; some application must route it.
Except for default input and output, Windows 11 sound devices are configured for exclusive access.
Consequently, any not-default input (e.g. text-to-voice) output must be somehow split
to feed both e.g. Somnium Space VR and default output.
System > Sound > Sound Mixer, where Sound Mixer requires scrolling nearly to the bottom of System > Sound.
The v to the right of each App exposes Output and Input device selection.
