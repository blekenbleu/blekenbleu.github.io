Windows VST Host

For use with SimHub and VoiceMeeter ASIO Insert,
a VST Host must support MIDI and 8 channel ASIO.
Few support all VST plugins flavors (VST or VST3, 32- or 64-bit).
VST hosts mentioned in VB-Audio's Forums:
Cantabile Lite, Carla, Element, LightHost, VSTHost
