Windows 11 considerations
  • using Visual Studio 2022 Community
  • added CMake and path to it
  • added path to MSBuild.exe (VS/2022/Community/MSBuild/Current/Bin)
    and bash alias msbuild=MSBuild.exe
  • C:\Program Files\Common Files\VST3\

Building VST3 using msys64 ucrt64.exe, from How to build VST3:
  • created C:\Users\bleke\AppData\Local\Programs\Common\VST3\
  • temporarily enable Developer Mode in Settings > System

Using MSYS2 ucrt64.exe: 
$ cd /d/my/GitHub/
$ mkdir VST3
$ cd VST3
$ git clone --recursive https://github.com/steinbergmedia/vst3sdk.git
$ mkdir build
$ cd build
$ cmake.exe -G "Visual Studio 17 2022" -A x64 ../vst3sdk
$ cmake --build .
