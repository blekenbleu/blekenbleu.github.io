Intel Extreme Tuning Utility (Intel XTU)

click for details
  • Intel seems best positioned to understand overclocking their CPUs...
  • Easy tweaks in Windows without semi-permanent boot changes
  • Sadly, some controls seem doomed, e.g. non-Default Core Voltage with Adaptive
  • Stress applies higher Voltages than Benchmark and Tunings
  • Profiles can be saved but not loaded???
    Highlighting e.g. Default in Profiles, then Show Values makes them Proposed, then Apply to make Active.
  • Default stress open covers MOS peaked at 46.5C before 5 minutes; CPU thermally throttled with ~1.37V core
    Core Max, CPU Package, MOS and (DDR5) SPD Hub Temperatures increase during stess test
  • 5387mHz 1255mV -0.095 Core Offset Voltage passed stress;@nbsp -0.10V failed...
  • XTU Memory Stress test provoked CPU thermal throttling with -95mV offset when CPU stress did not...
Covers raised MOS (45-48) and DDR5 Temperatures (41-44) about 3C during XTU stress testing.

MSI Center User Scenario Customize Fan
click for details
  • MSI Center fan tuning:  Features > User Scenario > (Fan on) > Customize (clover)


    CPU Fan


    Smart Fan1


    Smart Fan5:  negligible heat exhausted by Antec rear...


  • Smart Fan6 Antec controller poorly matches Arctic case fans,
    running fast at very low PWM % needed for quiet low load operation,
    still overwhelming Fan5, with essentially all heat out case top

  • Arctic case fans then need to be driven by CPU temp to exhaust warm air
  • Arctic fans overwhelm Antec rear exhaust fan.
  • Quieter and more powerful Scythe Kaze Flex 120mm PWM ordered to replace stock Antec exhaust
  • CPU fan (ID-COOLING SE-226-XT) seems fine;  fin stack top is always cool...
  • Antec rear exhaust fan is noisy;  idle speed reduced to be tolerable
  • Driving Antec chamber fan from GPU temperature is supported by neither MSI BIOS nor Center.

  • 24 Aug 2023:  KazeFlex 120 PWM fan received, replacing rear Antec fan
        reconfigured SYSFan1 to PWM in BIOS;
        dialed new SYSFan1,5,6 curves in MSI Center User Scenario for reduced idle noise    
    Fan1:  now Scythe KazeFlex PWM

     
    Fan5

     
    Fan6 :  replacing puny stock Antec fan with Scythe's KazeFlex significantly altered airflow;
    most heat now flows out the back instead of the top, with less noise.  Case air pressure
    may now be negative, wanting top exhaust fans driven separately from front intake fans.
