Voicemeeter package includes Virtual Audio I/O (Audio Device driver)
-----------------------------------------------------------------

Voicemeeter Virtual Mixing Console is Programmed by V.Burel
Copyrights V.Burel©2010-2024 All Rights Reserved.

Voicemeeter Potato includes VB-CABLE technology to provide 3x virtual 
audio i/o. VB-CABLE is an Audio Driver working as virtual audio cable.

Voicemeeter release in 2024 is installing a single audio driver on 
Win10/11 x64 and Arm64 providing 8 virtual I/O:
   - the 3 usual for potato
   - 5 VAIO extensions
more info: https://voicemeeter.com/voicemeeter-updates-february-2024/

Voicemeeter package also installs: 
   - up to 5x ASIO Drivers (32 and 64 bits)
   - Voicemeeter Remote API (32 and 64 bits)
   - Macro Button application
   - VBAN2MIDI application
   - 15 Band Graphic EQ APA for Voicemeeter BUS
   - 8x8 Gain Matrix APA for Voicemeeter BUS
   - Voicemeeter Streamer View Application
   - VBDeviceChecker application

-----------------------------------------------------------------
INSTALLATION
-----------------------------------------------------------------

Just run the VoicemeeterBananaSetup program (! IN ADMINISTRATOR MODE !).

REBOOT Must be done after installation (and after Uninstallation).
