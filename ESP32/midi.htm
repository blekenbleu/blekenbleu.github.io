Known Issue with current Arduino ESP32TinyUSB library:
"tinyusb callbacks cant be overriden and are used by arduino-esp32"
Most Arduino MIDI examples want hacking for ESP32-S2
if only for digitalWrite(LED_BUILTIN, HIGH);,
which LED_BUILTIN does not exist.
Instead, downloaded https://www.arduino.cc/reference/en/libraries/freenove-ws2812-lib-for-esp32/ and created esp32s2LED library to minimize LED substitution hassles.
