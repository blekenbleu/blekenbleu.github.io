TinyUSB Geneology

adafruit ported TinyUSB to ESP32-S2 for Metro and FeatherS2 products,
mostly with CircuitPython but also the Arduino IDE.

There are now also ESP32-S2 boards from LILYGO, SparkFun, UnexpectedMaker
  • LILYGO provides absolutely minimal documentation, none useful for Arduino.
  • SparkFun's hookup guide includes the Silicon Labs' Windows VCP driver for Arduino loading.
  • UnexpectedMaker describes using esptool to flash ESP32-S2.
Arduino TinyUSB libraries include
Adafruit TinyUSB Library
ESP32TinyUSB
Both link to GitHub for releases
adafruit has https://github.com/adafruit/Adafruit_TinyUSB_Arduino and TinyUF2 Bootloader
  Their ESP32 port depends on Espressif's esp32-hal-tinyusb.c for building USB descriptors.
  It includes examples updated to work with esp32s2
  adafruit also has a webpage guide
ESP32TinyUSB is the chegewara / EspTinyUSB implementation with seemingly useful example sketches..
