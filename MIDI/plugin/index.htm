SimHub MIDI Plugins

SimHub is largely a collection of so-called plugins, e.g. one for each supported game,
then others for SimHub-provided Options enabled under Settings.
User-provided plugins can be enabled via Additional Plugins.
Plugins can interact via Properties, Actions and Events, where Actions (e.g. button presses) may launch Events.
Properties can contain numeric or string variables.
Instead of requiring code to check for changing values e.g. 60 times per second,
one could presumably provoke an Event to launch code in an Action to evaluate changed Properties.
Packaged as .dll files, SimHub plugins are installed by dragging into
the SimHub folder containing SimHubWPF.exe and numerous other .dll files.
SimHub presumably identifies plugins from other .dll files by including this line in their source:
  using SimHub.Plugins;

C# expressions
SimHub property values can also be set using C# custom expressions.
SimHub support for these C# expressions preceded ncalc, JavaScript, plugin sdk, etc
and is kept for backward compatibility.
As of 7.04.8b4, these are evaluated a first time when loaded.
To avoid user interface coding, consider using a C# custom expression
for setting MIDI input and output device name property strings.
