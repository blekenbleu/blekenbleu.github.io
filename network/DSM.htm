Synology RAID Calculator: SHR vs. RAID

SHR compares somewhat to RAID 5:
  • if all drives are the same size, one gets used for protection
  • if one is larger, the amount by which it is larger than the smallest (RAID 5) or next largest (SHR) gets wasted
  • if one is smaller, that size is the only amount used from each drive by RAID 5
  • SHR protection size is lesser of 2 largest drives.

Synology NAS hard drive health
There are really no other tests other then S.M.A.R.T.,
which won't catch mechanical problems for instance.
The Storage Manager will show the SMART status of each disk.
Log into the web interface and go to Main Menu > Storage Manager > HDD Management.

Test Scheduler Option on this screen can also schedule a more in-depth SMART test.
