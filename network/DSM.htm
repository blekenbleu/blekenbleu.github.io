    Data scrubbing and "bit rot"

    Data scrubbing on BTRFS volumes first runs file system data scrubbing for data accuracy,
    then RAID scrubbing for data consistency.
    File system data scrubbing calculates a checksum for every written file in BTRFS volumes
    and further protects those data checksum with a metadata checksum.
    RAID 5 and SHR implement parity striping at the block level, with parity blocks distributed across blocks.
    Perform or Schedule Data Scrubbing.
