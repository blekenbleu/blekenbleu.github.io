Action

Get | Set move settings from | to the device.
Load | Save move settings from | to PC files.

Tools

Default is wanted for a new device or recovering from (mis)configuration.
Start Calibr | End Calibr are wanted to determine axes limits after Default.
Other "Tools" should be considered nearer weapons and avoided.

Settings

Options other than Auto Read Parameters, Short Device List
are presumably for bored or masochistic folks.

The bottom section

mostly about configuring VKB device control behaviors (e.g., for me) EVO R XT2

with actual significances rarely explained in single VKB PDF paragraphs
and too often never.  Some terms (dynamic, logical, virtual) are context-dependent...
