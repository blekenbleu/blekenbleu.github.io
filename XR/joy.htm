Five kinds of controllers

  • DirectInput (non-gamepad) e.g. HOTAS (joysticks), button boxes.
  • gamepad (PlayStation, XBox 360 (XInput), etc)
  • hand tracking and gestures (e.g. UltraLeap)
  • MIDI
  • motion (6 DoF VR):  sense position and angle for virtual laser pointing

SteamVR used to support DirectInput

  • Specifically, this HOTAS HOW-TO described

      select "Xbox One Controller" for the type"
    ... but neither popup nor option to select type now exists.

Other than motion controllers, SteamVR now supports only XInput

  • With XInput controllers, SteamVR 2.9.6 uses HMD gaze for pointing.
