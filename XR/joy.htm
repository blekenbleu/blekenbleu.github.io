joystick controller in SteamVR

Four kinds of controllers

  • MIDI
  • motion (6 DoF VR):  sense position and angle for virtual laser pointing
  • gamepad (PlayStation, XBox 360 (XInput), etc)
  • non-game (AKA DirectInput) e.g. HOTAS (joysticks), button boxes.

SteamVR used to support DirectInput and XInput,
  but now seemingly only XInput.

  • Specifically, this HOTAS HOW-TO described
    "7. On the popup, give the controller an easy label like "My NMS Joystick";
      select "Xbox One Controller" for the type"
    ... but neither popup nor option to select type now exists.
  • SteamVR 2.9.6 does still support XInput controllers, using HMD gaze for pointing.
  • HOTAS is IMO far easier than emulating motion controllers by hand gestures.
  • HOWTO: Make a gamepad work with SteamVR

XInput emulation

Several XInput emulators employ ViGEm kernel-mode driver
that obsoleted SCP Virtual Bus Driver.
