Hand-On Throttle-And-Stick (HOTAS) as VR controller
    Assetto Corsa in Odyssey+ unavoidably involves Windows Mixed Reality Portal and SteamVR,
    which support only controllers that I find intolerable and make no sense when sim driving.
    Without a controller, using 2-D apps such as Content Manager in HMD is seemingly impossible,
    except perhaps with a virtual desktop app.
    G29 rim buttons become unavailable when using AccuForce.
    If more than a couple of controller buttons are needed, a flight joystick may be wanted.

  • In Steam > Settings (SteamVR must be turned off)
    > Controller, General controller settings.
    Uncheck all but Xbox Configuration Support
    and Xbox Extended Feature Support.
  • Go to detected controllers, customize HOTAS layout.  Save.
  • Open SteamVR; a gamepad icon should appear.
    If not, be sure to open SteamVR by clicking VR button in Steam.
  • Make applicable mappings from virtual gamepad in SteamVR app.
    TIPS:
    If HOTAS Throttle doesnâ€™t work or is mapped upside down,
    clear the input and map it to W and S keys with a joystick profiler.
    Make sure no keys are pressed while launching the game.
  • Microsoft SideWinder Precision 2 USB Controller Joystick seems likely as a HOTAS.
  • Universal Control Remapper AKA UCR allows remapping inputs from devices
    such as second mice (using interception) and joysticks to virtual HID game controllers.
    • remap HOTAS using UCR and VIGEm
    • UCR Wiki
    • join multiple controllers
  • Steam's Big Picture interface has to be told not to use the gamepad for its own interface.
    Big Picture and SteamVR are 2 competing interfaces to the Steam content,
    and only one of them can "own" the gamepad.
