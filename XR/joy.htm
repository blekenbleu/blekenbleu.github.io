Four kinds of controllers

  • MIDI
  • 6DoF VR AKA motion:  sense position and angle, support virtual laser pointing
  • gamepad (PlayStation, XBox 360 (XInput), etc)
  • non-game (AKA DirectInput) e.g. joysticks, button boxes.

SteamVR used to support DirectInput and XInput,
  but now seemingly only XInput.

  • SteamVR supports XInput controllers, using HMD gaze for pointing.
  • This is IMO far easier than emulating VR controllers by hand gestures.
  • HOWTO: Make a gamepad work with SteamVR

XInput emulation

Several Xinput emulators employ ViGEm kernel-mode driver
that obsoleted SCP Virtual Bus Driver.

    Steam Input Devices - typically different from SteamVR controllers
  • In Steam -> Settings (SteamVR must be turned off)
    -> Controller, General controller settings.
    Uncheck all but Xbox Configuration Support
    and Xbox Extended Feature Support.
  • Go to detected controllers, customize HOTAS layout.  Save.
  • Open SteamVR; a gamepad icon should appear.
    If not, be sure to open SteamVR by clicking VR button in Steam.
  • Make applicable mappings from virtual gamepad in SteamVR app.
