    A brief history of Audio-Technica's 60 years

    VM cartridge history
    • Cartridge-making Dictionary
    • Audio Technica founded 17 Apr 1962
    • 1962:  AT-1 moving magnet with diamond needle in uncrushed cantilever tube, higher end AT-3
    • stereo AT-5
    • 1966:  Moving Pulse test record
    • 1967:  AT-35X, first 'VM' type dual moving magnet stereo cartridge
    • VM cartridge features and mechanism
    • AT-VM3:  first mass-production using precision moulds;
      hard duralmin cantilever only 50 microns thick
    • AT-VM35:  tapered cantilever
    • AT-VM35F:  four-channel wideband playback over 45kHz
    • 1978 AT25:  VM cartridge with toroidal power system
    • 1979 AT120E/G:  low-loss paratoroidal power system with a magnetic shield
      between channels, each with Omega (broken circle) toroidal coils in parallel
    • 1986:  AT=ML150/OCC low oxygen long crystal copper wiring
    • AT33E
    • 1987:  AT33ML using single-crystal high-purity oxygen-free copper
    • 1997:  AT150MLX
    • 2011:  AT150ANV sapphire cantilever
    • AT-VM95 vs AT95E:  VM95 has greater frequency response and higher output,
      threaded mounting inserts
    • AT-VM vs AT-VM95:  VM uses paratoroidal coil construction
      (two coils per channel - one stacked behind the other) with OFC wire
    • VM500 vs VM700:  700 has die-cast aluminum-alloy housing and tapered cantilever;
      all use 99.9999% pure oxygen free copper wiring
